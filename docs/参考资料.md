以下按女频、男频、脑洞/悬疑、现实向四大类，整理番茄小说 2 万字以下（1-2 万字）、高阅读、高完读的短篇爆款案例，包含题材、书名、主要内容、核心冲突、爽点/钩子、字数，全部是平台验证过的高热模板，直接套用即可。

**2 万字内爆款最多，情绪强、完读率高**

### 1. 追妻火葬场 / 虐渣复仇（女频 Top1）

**题材**：现代豪门/都市、虐恋 + 爽文、短篇复仇

**书名公式**：极端场景 + 背叛 + 反转（ICU、车祸、绝症、离婚、葬礼）

**代表书名**：

| 书名                      | 字数    |
| ----------------------- | ----- |
| ICU 门外，他抱白月光笑时，我心电图成了直线 | 1.2 万 |
| 离婚当天，前夫跪我公司楼下求复合        | 1.5 万 |
| 葬礼上，重生的我笑看前夫悔断肠         | 1.8 万 |
| 孕晚期被推下楼，重生后我让他净身出户      | 1.6 万 |

**主要内容**（通用模板）：

女主前期深爱男主，被男主 + 白月光联手伤害（流产、残疾、名誉尽毁、家破人亡），心死后离婚/死亡/重生，男主失去后才追悔，疯狂追妻，女主冷漠打脸，最终事业爱情双丰收（或独美）。

**核心冲突**：

- **前期**：深情女主 vs 冷漠渣男 + 白月光陷害
- **中期**：心死女主 vs 幡然醒悟疯狂追妻的前夫
- **后期**：女主复仇搞事业 vs 渣男阻挠 + 后悔

**爽点**：重生/觉醒后智商在线、虐渣干净利落、男主卑微追妻、女主独美搞钱。
