## 钩子类型

### 1. 悬念钩子

**定义**：留下悬念，引发读者猜测

**特点**：
- 信息不完整
- 有明确的疑问点
- 强烈的好奇心驱动

**示例**：
```
她重生了。就在她死亡前的那一天。
```
```
林婉手里拿着一份文件，陆沉脸色瞬间变了。
```
```
"其实，你真正应该找的人，不是我。"
```

**强度评估因素**：
- 悬念清晰度（9.2）
- 与剧情关联度（9.0）
- 震撼程度（8.8）

---

### 2. 反转钩子

**定义**：意外转折，打破预期

**特点**：
- 出人意料
- 逻辑自洽
- 引发思考

**示例**：
```
原来他不是出轨，而是去调查她的仇人。
```
```
苏月微笑着说："其实，我才是真正的受害者。"
```
```
陆沉看着林婉，突然跪下了。
```

**强度评估因素**：
- 意外性（9.5）
- 合理性（8.8）
- 影响范围（9.0）

---

### 3. 危机钩子

**定义**：突发危机，创造紧张感

**特点**：
- 时间紧迫
- 危险明确
- 必须解决

**示例**：
```
手术室外，医生缓缓摇头。
```
```
"对不起，林总，您被解雇了。"
```
```
远处的警笛声越来越近。
```

**强度评估因素**：
- 危机严重程度（9.3）
- 时间紧迫感（9.0）
- 解决难度（8.5）

---

### 4. 打脸钩子

**定义**：爽点打脸，满足期待

**特点**：
- 反派受打击
- 主角展现实力
- 爽感拉满

**示例**：
```
陆沉跪在雨中：婉婉，求你回来。
```
```
全场掌声雷动，苏月的脸色惨白。
```
```
林婉淡淡地说："从今天起，这家公司归我了。"
```

**强度评估因素**：
- 打脸程度（9.4）
- 反派反应（9.0）
- 旁观者反应（8.8）

## 钩子强度分级

| 等级 | 评分 | 标准 | 示例 |
|------|------|------|------|
| 极强 | 9.5+ | 震撼、意外、高度期待 | "她重生了。就在她死亡前的那一天。" |
| 强 | 8.5-9.4 | 明确悬念、强烈好奇 | "林婉手里拿着一份文件，陆沉脸色瞬间变了。" |
| 中 | 7.5-8.4 | 有悬念、期待一般 | "明天就是婚礼了。" |
| 弱 | <7.5 | 悬念模糊、期待不足 | "她笑了笑。" |

## 钩子分配策略

### 按章节范围分配

| 章节范围 | 推荐钩子类型 | 原因 |
|----------|--------------|------|
| 第1-5章 | 悬念 > 反转 > 危机 | 建立悬念，吸引读者 |
| 第6-15章 | 反转 > 打脸 > 悬念 | 推进剧情，增加爽点 |
| 第16-19章 | 危机 > 打脸 > 反转 | 高潮对决，紧张激烈 |
| 第20章 | 无 | 结局，无需钩子 |

### 按剧情阶段分配

**前期压迫（第1-5章）**：
- 悬念：引发读者好奇心
- 反转：初步打脸
- 危机：建立紧迫感

**中期反转（第6-15章）**：
- 反转：剧情转折
- 打脸：爽点密集
- 悬念：维持期待

**后期爽点（第16-19章）**：
- 危机：高潮对决
- 打脸：终极打脸
- 反转：意外结局

## 钩子设计技巧

### 1. 位置控制
- **位置**：章末 50-100 字
- **避免**：不要在段中间插入钩子

### 2. 长度控制
- **长度**：1-3 句
- **避免**：过长钩子影响节奏

### 3. 信息控制
- **悬念钩子**：只给信息的一半
- **反转钩子**：打破预期但不解释
- **危机钩子**：只呈现危机不揭示结果

### 4. 情绪控制
- 前期：悬念引发好奇
- 中期：反转引发期待
- 后期：危机引发紧张

## 钩子质量检查

### 检查清单

| 检查项 | 要求 |
|--------|------|
| 钩子位置 | 章末 50-100 字 |
| 钩子类型 | 悬念/反转/危机/打脸 |
| 钩子强度 | 评分 >= 8.0 |
| 与剧情关联 | 高度相关 |
| 逻辑自洽 | 不前后矛盾 |
| 读者期待 | 有强烈继续阅读欲望 |

### 不合格钩子示例

**钩子过弱**：
```
她笑了笑。
```
问题：悬念模糊，期待不足

**钩子过长**：
```
林婉看着窗外的夜景，心中五味杂陈。她想起以前和陆沉的点点滴滴，想起那些甜蜜的时光，也想起那些痛苦的回忆。她知道自己已经不能回头了，所以她决定...
```
问题：过长，影响节奏，无明显悬念

**钩子无关联**：
```
今天天气真好。
```
问题：与剧情无关，无期待感

## 注意事项

1. 每章必须有钩子（第20章除外）
2. 钩子强度评分 >= 8.0
3. 钩子位置在章末 50-100 字
4. 钩子类型合理分配
5. 与下章开篇高度关联
6. 逻辑自洽，不前后矛盾
7. 引发强烈继续阅读欲望
