# 冲突检查

## 职责

确保核心冲突贯穿全文，三要素分布合理，剧情逻辑严密。

## 输入格式

```json
{
  "full_draft": {
    "chapters": [
      {
        "chapter_number": 1,
        "content": "章节内容...",
        "word_count": 856
      },
      ...
    ]
  },
  "core_conflict": {
    "early_stage": "前期：深情女主 vs 冷漠渣男 + 白月光陷害",
    "mid_stage": "中期：心死女主 vs 幡然醒悟疯狂追妻的前夫",
    "late_stage": "后期：女主复仇搞事业 vs 渣男阻挠 + 后悔"
  },
  "reference_path": "../docs/参考资料.md"
}
```

## 核心冲突三要素

### 1. 前期压迫（第1-5章）

**目的**：建立冲突起点，引发读者共鸣

**包含元素**：
- 主角被伤害/轻视
- 反派嚣张
- 逆境爆发

**检测标准**：
| 检查项 | 要求 |
|--------|------|
| 冲突出现时机 | 第1章300字内 |
| 冲突强度 | 明显/强烈 |
| 受害程度 | 让读者共情 |
| 反派行为 | 令人反感 |

---

### 2. 中期反转（第6-15章）

**目的**：剧情转折，主角觉醒反击

**包含元素**：
- 主角觉醒/重生
- 反转剧情
- 初步打脸

**检测标准**：
| 检查项 | 要求 |
|--------|------|
| 转折清晰度 | 明确的觉醒点 |
| 反转强度 | 意外/震撼 |
| 打脸频率 | 至少3次 |
| 上升态势 | 持续增强 |

---

### 3. 后期爽点（第16-20章）

**目的**：高潮决战，终极打脸，完美结局

**包含元素**：
- 高潮对决
- 终极打脸
- 反派崩溃/后悔
- 主角胜利

**检测标准**：
| 检查项 | 要求 |
|--------|------|
| 高潮强度 | 最强点 |
| 打脸程度 | 爽快拉满 |
| 反派反应 | 崩溃/后悔/跪舔 |
| 结局满意度 | 逻辑闭环，爽感拉满 |

## 冲突映射

### 映射流程

1. **提取冲突关键词**
   - 识别压迫场景（第1-5章）
   - 识别反转场景（第6-15章）
   - 识别打脸场景（第16-20章）

2. **匹配核心冲突**
   - 检查压迫冲突是否完整
   - 检查反转冲突是否清晰
   - 检查爽点冲突是否充分

3. **评估冲突强度**
   - 计算各阶段冲突强度
   - 识别薄弱环节
   - 评估整体效果

### 强度评估

| 等级 | 强度评分 | 标准 |
|------|----------|------|
| 极强 | 9.0+ | 冲突明显、强烈、共鸣强 |
| 强 | 8.0-8.9 | 冲突清晰、有效 |
| 中等 | 7.0-7.9 | 冲突存在但不够强 |
| 弱 | <7.0 | 冲突模糊、缺失 |

## 缺失冲突检测

### 检测场景

1. **冲突缺失章节**
   - 章节中无冲突情节
   - 只有描述性内容
   - 缺乏对抗性

2. **冲突强度不足**
   - 有冲突但力度不够
   - 反派行为过于温和
   - 主角反应不强烈

3. **冲突类型单一**
   - 全程只有一种冲突
   - 缺乏层次和变化
   - 重复同一模式

### 检测方法

```json
{
  "chapter_analysis": {
    "chapter": 8,
    "conflict_detection": {
      "has_conflict": true,
      "conflict_type": ["对话冲突", "心理冲突"],
      "conflict_strength": 5.5,
      "issue": "冲突强度不足，建议增强"
    }
  }
}
```

## 冲突轨迹

### 轨迹分析

分析冲突的演进轨迹：

```
压迫 → 觉醒 → 反击 → 升级 → 高潮 → 结局
  ↓      ↓      ↓      ↓      ↓      ↓
第1章  第6章  第10章  第15章  第18章  第20章
```

### 轨迹检查

| 检查项 | 要求 |
|--------|------|
| 起点明确 | 第1章冲突清晰 |
| 转折自然 | 觉醒点合理 |
| 上升持续 | 强度持续增强 |
| 高潮明显 | 第18章冲突最强 |
| 结局完整 | 逻辑闭环爽感足 |

## 修正建议

### 基于缺失冲突

```json
{
  "suggestions": [
    {
      "chapter": 8,
      "type": "add_conflict",
      "suggestion": "建议增加女主与陆沉的对话冲突，突出心死后的冷漠",
      "example": "\"陆沉，我们已经结束了。\"林婉冷冷地说。"
    }
  ]
}
```

### 基于强度不足

```json
{
  "suggestions": [
    {
      "chapter": 12,
      "type": "strengthen_conflict",
      "suggestion": "建议增强打脸强度，增加旁观者反应",
      "example": "周围的人纷纷议论："陆大少这是报应啊...""
    }
  ]
}
```

## 评分标准

```json
{
  "score_breakdown": {
    "early_stage_presence": 9.5,     // 前期压迫完整性
    "mid_stage_presence": 9.0,       // 中期反转完整性
    "late_stage_presence": 9.3,      // 后期爽点完整性
    "conflict_trajectory": 9.2,       // 冲突轨迹合理性
    "overall_score": 9.3              // 综合评分
  }
}
```

| 分数 | 冲突质量 | 说明 |
|------|----------|------|
| 9.0+ | 优秀 | 冲突完整、强度高、轨迹清晰 |
| 8.0-8.9 | 良好 | 冲突基本完整，个别章节需微调 |
| 7.0-7.9 | 一般 | 存在明显缺失，需加强 |
| <7.0 | 较差 | 冲突混乱、缺失严重，需大幅修正 |

## 注意事项

1. 三要素必须完整呈现
2. 冲突强度需持续增强
3. 转折点必须清晰
4. 高潮必须最强
5. 评分需客观准确
