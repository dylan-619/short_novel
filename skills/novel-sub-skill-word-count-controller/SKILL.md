# 字数控制

## 【执行模式】单章节处理

**【关键】本技能必须**：
- 每次只处理一个章节
- 等待字数调整完成后才返回结果
- 不得批量处理多个章节
- 不得跳过字数控制

## 职责

精确控制每章字数在配置的范围内，确保符合目标平台标准。支持灵活的字数范围配置：
- **短视频剧本**：200-300字/章
- **短篇爆款**：800-1000字/章
- **中篇小说**：1000-1500字/章
- **长篇小说**：1500-3000字/章

## 调整策略

### 1. 精简策略（字数过多）

**优先删除内容**（按重要性排序）：

1. **水文描写**（最不重要）
   - 过长环境描写
   - 重复的心理活动
   - 无意义的表（情描写
   - 冗余的过渡段落

2. **冗余表达**
   - 重复的词句
   - 过度的修饰
   - 冗长的对话

3. **次要细节**
   - 非核心情节细节
   - 可删减的场景
   - 可合并的段落

**精简方法**：

| 方法 | 适用场景 | 示例 |
|------|----------|------|
| 删除水文 | 无关环境描写 | 删除"窗外的雨..." |
| 精简对话 | 冗余对话 | "我...我没事" → "我没事" |
| 合并段落 | 内容重复 | 合并相似段落 |
| 删除冗余 | 重复表达 | 删除重复的心理描写 |

**精简示例**：
```
原句：窗外的雨淅淅沥沥地下着，打在玻璃上发出清脆的响声。路灯昏暗，照在湿漉漉的街道上，远处偶尔传来汽车驶过的声音。

精简：窗外雨下个不停，远处传来汽车驶过的声音。
```

---

### 2. 扩充策略（字数不足）

**优先扩充内容**（按重要性排序）：

1. **核心情节细节**
   - 对话细节
   - 心理活动
   - 动作描写

2. **环境氛围**
   - 短暂环境描写
   - 氛围渲染

3. **次要情节**
   - 旁观者反应
   - 环境细节
   - 人物互动

**扩充方法**：

| 方法 | 适用场景 | 示例 |
|------|----------|------|
| 扩充对话 | 增加冲突 | 增加反挑衅话术 |
| 增加心理 | 突出人物 | 增加内心活动 |
| 增加细节 | 丰富场景 | 增加环境细节 |
| 增加反应 | 强化效果 | 增加旁观者反应 |

**扩充示例**：
```
原句：林婉看着他们，突然笑出声来。

扩充：林婉看着他们，突然笑出声来。笑得眼泪都出来了，笑得浑身发抖。她没想到，自己爱了三年的男人，竟然会这样对她。
```

---

### 3. 微调策略（字数接近）

**微调方法**：

| 场景 | 方法 |
|------|------|
| 字数稍多 | 删除 1-2 个词 |
| 字数稍少 | 增加 1-2 个形容词 |
| 字数接近 | 保持原样 |

## 字数计算

### 计算方法

```python
def count_chinese_words(text):
    """计算中文字符数（不包含空格和换行）"""
    import re
    # 提取中文字符
    chinese_chars = re.findall(r'[\u4e00-\u9fff]', text)
    return len(chinese_chars)
```

### 计算示例

```
内容："ICU 门外，林婉站了整整三天。"
中文字符数：12（ICU 门外，林婉站了整整三天）
```

## 调整限制

### 优先级保护

**最高优先级（不可删除）**：
- 核心情节
- 章末钩子
- 关键对话
- 冲突高潮

**中等优先级（可微调）**：
- 心理描写
- 动作描写
- 环境氛围

**最低优先级（可删除）**：
- 水文描写
- 冗余表达
- 次要细节

### 调整次数

| 期望 | 最多调整次数 |
|------|--------------|
| 完全达标 | 5 次 |
| 接近达标 | 10 次 |

## 注意事项

1. 字数必须控制在 target_min 和 target_max 范围内（不再固定为 800-1000）
2. 优先保护核心情节和钩子
3. 删除水文不影响剧情逻辑
4. 扩充不影响节奏紧凑
5. 最多调整 10 次
6. 保留人物性格一致性
7. 记录所有修改便于审核
8. 支持各种字数范围（200-300, 400-600, 800-1000, 1000-1500, 1500-3000等）
